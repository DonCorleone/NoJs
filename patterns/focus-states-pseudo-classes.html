• :focus — fires on every focus method: click, tab, programmatic
  • :focus-visible — browser shows ring only when it helps (keyboard nav, not mouse clicks)
  • :focus-within / :has(:focus-visible) — parent reacts to a focused child, with or without the mouse-click caveat
:focus

Ring appears on every focus method — click, tab, programmatic.

:focus-visible

Ring appears for keyboard only. Click won't show it.

:focus-within

Parent highlights when any child is focused — any method.

:has(:focus-visible)

Parent highlights for keyboard only — the missing "focus-visible-within".
