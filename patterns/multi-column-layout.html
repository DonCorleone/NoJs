• column-count — number of columns; browser handles all line breaking
  • column-fill: balance — equalises column heights automatically
  • column-gap — gutter between columns, same unit as grid/flex gap

The column-count property has been in every major browser since 2011, yet most developers still reach for flexbox or grid the moment they need side-by-side content.

With column-fill: balance the browser automatically distributes text so each column ends at roughly the same height — no JavaScript, no manual height calculations, no layout shifts.

column-gap works exactly like its grid counterpart, giving you a clean gutter between columns. Add column-rule if you want a visible divider line too.

Resize the window and watch the text reflow. The browser handles everything: line breaking, orphan control — all without a single line of script.
