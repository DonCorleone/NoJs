• clamp(min, preferred, max) — fluid value between a fixed floor and ceiling
  • the preferred value is usually a vw-based expression so it scales with the viewport
  • works on any numeric property — font-size, padding, gap, width, margin…
font-size: clamp(1rem, 0.3rem + 3.47vw, 2rem)
Resize the window to see me scale.
padding: clamp(8px, 2vw, 32px)
padding shrinks and grows with the viewport
gap: clamp(8px, 3vw, 48px)
item item item
width: clamp(120px, 50%, 400px)
never narrower than 120px, never wider than 400px
