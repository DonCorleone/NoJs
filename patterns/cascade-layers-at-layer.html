• @layer base, components, utilities — one declaration sets the paint order forever
  • all three selectors have identical specificity (1 class) — layer position alone decides the winner
  • without @layer, source order decides — whoever is written last wins, fragile and accidental
without @layer — source order wins
utilities — first in source, bottom ✗
components — middle
base — last in source, top ✗
with @layer — layer order wins
base — first layer, always bottom ✓
components — middle ✓
utilities — last layer, always top ✓
